# API (Application Programming Interface)

> 쉽게 말해 누군가가 미리 만들어둔 인터페이스를 이용할 수 있는 것
> 

EX) System Interface의 System.out.println → 출력 메소드
<br>

### 1.  java.lang 패키지

> Object, System, String, Math 등의 클래스
> 
- `import`없이 사용 할 수 있다.
<br>

### 2. java.util 패키지

> 대부분이 Collection 클래스
> 
- util 패키지는 `import`를 사용해야 한다.
<br>
