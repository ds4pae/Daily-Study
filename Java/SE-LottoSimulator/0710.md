# SE - lotto - Simulator

1번 선택

- [x]  시뮬레이션 할 복권 수(`count`) 입력
- [x]  랜덤 당첨 번호 7개 생성 (중복 X)
- [x]  자동 생성 번호 `count`개 만큼 생성
- [ ]  랜덤 당첨 번호 와 자동 생성 번호를 비교하여 등수 책정
- [x]  시뮬레이션이 끝나면 1,2 선택 화면 재출력
- [ ]  **++추가)** 당첨 번호와 자동 생성 번호를 올림차순으로 정렬

```
**** Selab Lotto Simulator ****
1. 시작
2. 종료
메뉴를 입력하시오>  1
시뮬레이션 할 복권 수를 입력하시오> 2
*********** 당첨 번호 ***********
10 15 23 27 37 9 + 29
********* 자동 생성 번호 *********
23 34 37 40 41 8
13 26 27 29 40 43
**** Selab Lotto Simulator ****
1. 시작
2. 종료
메뉴를 입력하시오> 2
************* END *************
```
